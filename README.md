# Personal Assistant Flutter App

## Create repo
1. Create a new GitHub repository.
2. Clone it locally.

## Upload code
1. Copy all files from this project into your repository.
2. Commit files.

## Push to main
```bash
git add .
git commit -m "Initial personal assistant app"
git branch -M main
git push -u origin main
```

## Go to Actions tab
1. Open your repository on GitHub.
2. Click **Actions**.
3. Open the **Build APK** workflow run.

## Download APK from Artifacts
1. Scroll to **Artifacts**.
2. Download **assistant-apk**.
3. Extract and install `app-release.apk`.
